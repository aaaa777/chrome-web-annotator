# Privacy Policy — Web Annotator

**Last updated**: 2026-04-08

## Data Collection

Web Annotator does **not** collect, transmit, or share any user data.

## Data Storage

All annotation data (selected text, comments, page URLs) is stored locally in your browser using `chrome.storage.local`. This data:

- Never leaves your device
- Is not sent to any external server
- Is not shared with any third party
- Is fully under your control

## Data Deletion

You can delete your annotation data at any time:

- **Individual annotations**: Click the delete button next to any annotation
- **Per-page**: Use the "このページを削除" button in the popup
- **All data**: Use the "すべて削除" button in the popup

Uninstalling the extension will also remove all stored data.

## Permissions

| Permission | Purpose |
|-----------|---------|
| `storage` | Store annotations locally in your browser |
| `activeTab` | Access the current tab to add/display annotations |

## Contact

For questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/aaaa777/chrome-web-annotator).
