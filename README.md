# Web Annotator

Webページにコメントを追加し、Markdown形式でエクスポートするChrome拡張機能。

テキストを選択してコメントをつけ、Markdown形式でクリップボードにコピーできます。

## 機能

- **テキスト選択 → コメント追加**: ページ上のテキストを選択すると、コメント入力フォームが表示されます
- **ハイライト表示**: 保存済みアノテーションを黄色マーカーで表示
- **ポップアップ一覧**: 現在のページ / 全ページのアノテーションを切り替えて閲覧
- **Markdownコピー**: アノテーションをMarkdown形式でクリップボードにコピー
- **データ管理**: 個別削除、ページ単位削除、全削除に対応
- **全ページ対応**: あらゆるWebページで動作

## インストール

### Chrome Web Store（審査中）

> 審査完了後にリンクを追加予定

### 手動インストール

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたディレクトリを選択

## 使い方

### コメントをつける

1. Webページ上でテキストを選択
2. 表示されるフォームにコメントを入力
3. 「保存」をクリック
4. テキストが黄色でハイライトされ、コメントが保存されます

### コメントを確認・コピーする

1. ツールバーの Web Annotator アイコンをクリック
2. 「このページ」「全ページ」タブでアノテーションを閲覧
3. 「コピー」ボタンでMarkdown形式でクリップボードにコピー

### コピー形式

```markdown
## アノテーション: [ページタイトル](URL)

> 選択したテキスト

コメント: 入力したコメント

---
```

## 技術仕様

- **Manifest**: V3
- **ストレージ**: chrome.storage.local（データはローカルのみ）
- **ビルド不要**: バニラJS、追加の依存関係なし
- **権限**: `storage`, `activeTab`

## プライバシー

本拡張機能はユーザーデータを外部に送信しません。すべてのアノテーションデータはブラウザのローカルストレージに保存され、端末内にのみ存在します。
